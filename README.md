# ⚡ AI Resume Analyzer

An AI-powered ATS resume scoring tool built using **pure HTML, CSS, and JavaScript** with TF-IDF cosine similarity — runs بالكامل in the browser with **no backend required**.

---

## 🚀 Features

- 📄 Upload PDF, DOCX, or TXT resume
- 📋 Paste any job description
- ⚡ Get an ATS compatibility score (0–100)
- 🔍 Keyword match & missing keywords analysis
- 📊 Skills breakdown with charts
- 💡 Actionable improvement suggestions
- 🔒 100% client-side (no data stored)

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript  
- **Charts:** Chart.js  
- **PDF Parsing:** PDF.js (CDN)  
- **NLP Algorithm:** TF-IDF + Cosine Similarity
